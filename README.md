# Rosachoque Instalações

## Site Profissional de Serviços Elétricos

Site responsivo e moderno para a empresa Rosachoque Instalações, especializada em serviços elétricos residenciais e comerciais.

## 📋 Características

- ✅ Design responsivo (mobile, tablet, desktop)
- ✅ Navegação suave com scroll automático
- ✅ Seção de serviços com cards animados
- ✅ Botão WhatsApp flutuante
- ✅ Atendimento 24h destacado
- ✅ Otimizado para SEO básico
- ✅ Performance rápida

## 🚀 Como Usar

### Localmente

1. Clone o repositório:
   ```bash
   git clone https://github.com/RUAYK/Rosachoque-instalacoes.git
   cd Rosachoque-instalacoes
   ```

2. Abra o arquivo `index.html` no seu navegador

### Online

Você pode fazer deploy do site em:

- **GitHub Pages** (grátis)
- **Vercel** (grátis)
- **Netlify** (grátis)
- **Heroku** (com plano pago)

#### Deploy no GitHub Pages:

1. Vá em **Settings** > **Pages**
2. Em **Source**, selecione **main** branch
3. Salve e aguarde alguns minutos
4. Seu site estará em: `https://RUAYK.github.io/Rosachoque-instalacoes`

## 📝 Editar Informações

### Telefone/WhatsApp

Edite em `index.html` (procure por `5511999999999`) e substitua pelo número real:

```html
https://wa.me/SEU_NUMERO_AQUI
```

Exemplo: `https://wa.me/5511998765432`

### Email e Endereço

Edite a seção "Contato" em `index.html`:

```html
<p><strong>Email:</strong> seu-email@seudominio.com.br</p>
<p><strong>Endereço:</strong> Sua cidade, Estado</p>
```

### Logo/Branding

Edite o nome em `index.html`:

```html
<div class="logo">
    Seu Nome
    <span>Sua Descrição</span>
</div>
```

### Cores

As cores principais estão em `style.css`:

- Azul escuro: `#071b33`
- Amarelo: `#ffc400`
- Verde WhatsApp: `#25d366`
